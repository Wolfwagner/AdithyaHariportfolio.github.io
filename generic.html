September 2023

PID-Controlled Trajectory Following and Control for Differential Drive Robots

This project focuses on controlling a differential drive robot to accurately follow a desired trajectory using PID controllers. The robot's kinematics, described in the inertial frame, involve regulating linear and angular velocities to minimize errors in angle and distance to the goal position. Components include implementing Euler integration for state updates, computing error metrics, utilizing PID controllers for velocity regulation, and defining desired trajectories. By optimizing PID controller parameters and output limits, the project aims to achieve precise trajectory following, enhancing the robot's capabilities for autonomous navigation and robotics research applications.

Advanced Sensor Fusion for Robotic Localization and Navigation

Exploring advanced techniques in robot localization, this project delves into the intricacies of sensor fusion for precise navigation. From modeling the limitations of MEMS Inertial Measurement Sensors to integrating GPS-based position measurements and encoder data, participants traverse a spectrum of scenarios. Through the implementation of Kalman filtering and Extended Kalman filtering techniques, the project explores the nuances of state estimation in robotics. Analyzing outputs such as position covariance, velocity covariance, bias covariance and trajectory comparisons gave insights into the challenges and solutions shaping modern robot navigation.

Geometry-Based Configuration for Robot Space Planning

Implemented a Python algorithm utilizing the NumPy and Pygame libraries, coupled with Configuration Space Planning, to automate the navigation of an 'alien' through a maze towards a predefined goal state. The maze layout necessitates the alien to adhere to waypoints, while also detecting maze boundaries and transforming into alternative shapes to maneuver through narrow passages. Employing a search strategy aimed at discovering the most cost-effective path to the goal node, the algorithm optimizes runtime efficiency by exclusively considering K-nearest neighbors.

September 2023

Pure Pursuit Algorithm for Autonomous Car

Developed a Python-based lateral and longitudinal controller capable of autonomously guiding a vehicle along a predefined racetrack within the ROS and Gazebo software environments. The longitudinal controller governs the vehicle's target speed by analyzing the track's curvature and the vehicle's current state based on upcoming waypoints. For instance, the speed is adjusted accordingly for straight paths or impending turns. Meanwhile, the lateral control utilizes a Pure Pursuit algorithm to determine the appropriate steering angle, considering factors such as the lookahead distance and the angle between the look-ahead line and the wheelbase. These controllers were designed based on the geometric principles of the Pure Pursuit algorithm.
