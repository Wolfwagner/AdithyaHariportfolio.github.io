Image Classification with neural networks

An image classification algorithm was developed using Python, NumPy, and PyTorch to categorize objects within images. Leveraging neural networks, non-linear activations, and multi-layer extensions of the linear perceptron, the algorithm was trained and validated on a CIFAR-10 dataset comprising RGB images with dimensions of 31x31 pixels. To enhance convergence and minimize loss, the algorithm utilized a ReLU activation function and L2 regularization techniques. The network architecture featured a minimum of four hidden layers, including Convolutional Neural Network (CNN) layers tailored for image recognition tasks. Upon testing with unseen images, the algorithm demonstrated an accuracy rate of 78%, showcasing its effectiveness in classifying objects within images.

POS Tagging with Hidden Markov Models: Viterbi Algorithms

Implemented Part-of-speech (POS) tagging using Hidden Markov Models (HMMs). Two tagging functions were implemented on the Brown dataset: a Baseline Tagger, which assigned tags based on the most frequent tag associated with each word during training. The accuracy of unseen words was 67%. The Viterbi Tagger, employed the Viterbi decoding algorithm to estimate tag probabilities based on transitions and emissions. This involved counting occurrences of tags and words, computing smoothed probabilities, constructing the Viterbi trellis, and back-tracing to find the best path. Laplace smoothing was used to handle unseen words and tags. The multi-tag accuracy was 94% while the unseen word accuracy was 64%.

Movie Review classifier : Naive Bayes approach

A binary sentiment classifier for movie reviews was built using the Naive Bayes algorithm. The dataset, comprising a large set of movie reviews, was processed to recommend only movies with positive reviews. The implementation included initializing with a bigram mixture model, where each bigram represented a sequence of two consecutive words from a review. The main task was to implement a function, which computed the mixture of unigram and bigram bag-of-words models. This involved calculating probabilities for both unigram and bigram models, combining them into a mixture model using a parameter Î», and tuning Laplace smoothing parameters to prevent zero probabilities. Experimentation with various parameters and preprocessing techniques, such as stemming, lowercase transformation, and stop word removal, was conducted to achieve an accuracy of 97%.

BFS search for Eightpuzzle and WordLadder problem

Devised the Best first search algorithm in Python to solve two different search problems: Eight Puzzle and Word Ladder. For the Word Ladder problem, a BFS algorithm was devised to iteratively explore neighboring states until the shortest path to the goal was determined, with a dictionary maintaining a record of visited states. For the Eight Puzzle problem was addressed using a BFS search approach enhanced by a Manhattan heuristic. Neighbors were generated for each state and defined a comparison method for tiebreaking in the priority queue. The addition of heuristics significantly improved the search efficiency.
